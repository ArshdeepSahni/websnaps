PROJECT DESCRIPTION
This project WebSnaps has been developed on HTML, CSS, JavaScript. The main objective of developing this website is to provide the user with a collection of 15 components in HTML i.e buttons, links et cetera. WebSnaps provides an user friendly environment for you to learn without any additional software to download. Here the user will able to navigate through the desired component along with its respective code called code snippets. It has a compatible responsive web design customized with a lot of CSS. The other highlights of this code include easy to use navigation bar, collapsible and responsive side bar and various icons and widgets.
